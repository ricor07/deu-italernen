_Un'applicazione desktop multipiattaforma sviluppata in **C++**, **Qt** e **CMake**._

- Interfaccia moderna con Qt  
  Moderne Benutzeroberfläche mit Qt  
- Compatibile con Windows, macOS e Linux  
  Kompatibel mit Windows, macOS und Linux

- C++17 o versione successiva  
  C++17 oder neuer  
- Qt 6.9.1
- CMake 3.14 o superiore  
  CMake 3.14 oder höher  
- Un compilatore C++ (GCC, Clang, MSVC…)  
  Ein C++-Compiler (GCC, Clang, MSVC…)
- Ollama (ultima versione se possibile) + modello gemma3n:e2b
- Ollama (letztes version wenn moglich) + model gemma3n:e2b

## 🛠️ Istruzioni per la compilazione / Build-Anleitung

### 1. Clonare il repository / Repository klonen

```
git clone https://github.com/ricor07/deu-italernen.git
cd deu-italernen
mkdir build
cd build
cmake ..
```

### 2. Compilare / Bauen

```make```

### 3. Eseguire / Starten
Su Linux/macOS:
Unter Linux/macOS:

```./deutschlernenApp```

Su Windows:
Unter Windows:

```deutschlernen.exe```
